# ShareIt - projektna dokumentacija

ShareIt je mobilna aplikacija, ki uporabnikom omogoča izposojo in deljenje predmetov (orodje, gospodinjski aparati, športna oprema ...), katerih nakup se ne splača za enkratno ali redko uporabo. Aplikacija povezuje posameznike v lokalnih skupnostih, spodbuja trajnostno uporabo virov in omogoča, da uporabniki z oddajo predmetov nekaj zaslužijo ali prihranijo.
